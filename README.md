# 赣南师范大学教务系统接口

仅供学习交流使用，请自觉遵守国家相关法律法规，谨慎使用，
任何因使用本项目造成的后果均与作者无关

本项目中所有接口均由分析前端请求得到，作者不对其稳定性做任何保证

## 依赖

本项目依赖于 `pytesseract` 库进行图片验证码识别，
请自行安装 `tesseract` 并修改 `login.py` 中的 `tesseract_cmd_path` 变量

如需使用其他库进行OCR请修改 `login.py` 中的 `recognize_captcha` 函数

## 项目结构

### 1. `api.py`

此文件为封装好的接口，如需调用api请尽量使用本文件中提供的接口

### 2. `login.py`

此文件为登陆相关的底层实现

### 3. `encode.py`

此文件为对统一登陆系统中加密函数的模拟
